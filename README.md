# ByBit Trading Bot

Автоматичний trading bot для криптобіржі ByBit з технічним аналізом, симуляцією та веб-інтерфейсом.

## Функціонал

- ✅ Реальна торгівля та симуляція (з віртуальними $100)
- ✅ Технічний аналіз (RSI, MACD, ковзні середні, смуги Боллінджера)
- ✅ Автоматичні прогнози та їх валідація
- ✅ Веб-інтерфейс для моніторингу
- ✅ Telegram бот для керування
- ✅ Детальне логування та БД
- ✅ Графіки з точками входу/виходу
- ✅ Зупинка/запуск/скидання стратегії
